# 🕒 LuxeTime – Modern Watch E‑Commerce Website

A modern, responsive luxury watch e‑commerce UI built with HTML, CSS, JavaScript, and Bootstrap, featuring glassmorphism cards, category-based filtering, and a minimal cart experience.

---

## 🔍 Features

- Responsive layout optimized for desktop, tablet, and mobile.
- Dark theme with **glassmorphism** styling and subtle 3D hover effects.
- Category tabs (All, Luxury, Sport, Classic) with client-side filtering.
- Featured products section above the fold.
- Simple cart badge and modal to simulate add‑to‑cart flow.
- Smooth scroll and light scroll‑triggered fade‑in animations.

---

## 🛠 Tech Stack

- HTML5
- CSS3 (custom styles, glassmorphism, 3D transforms)
- JavaScript (vanilla)
- Bootstrap 5 (grid, layout, utilities)
- Boxicons (icons)
- Google Fonts (Orbitron, Poppins)
